# Hassan Al Henaidi - Portfolio Website

A modern, responsive portfolio website with dark mode, GitHub API integration, and interactive project filtering.

## Quick Start

1. **Clone or download** this repository
2. **Open `index.html`** in any modern web browser
3. **No build steps or dependencies** required

## Features

- 🎨 **Dark/Light Mode** - Theme preference saved to localStorage
- 🔍 **Project Search** - Real-time filtering by title or description
- 🏷️ **Category Filter** - Filter projects by type (Web/App)
- 📊 **Sort Projects** - Sort alphabetically A-Z or Z-A
- 📱 **GitHub Integration** - Fetches live repositories (with demo fallback)
- 📝 **Contact Form** - Client-side validation with visual feedback
- 👋 **Dynamic Greeting** - Changes based on time of day
- 📈 **Visit Counter** - Tracks how many times you've visited

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- JavaScript (ES6+)
- GitHub REST API
- LocalStorage API
